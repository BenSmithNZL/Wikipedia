# Wikipedia
Script that shows the clusters Wikipedia pages based on the number of shared links.

## Example

![alt_text](https://github.com/BenSmithNZL/Wikipedia/blob/main/plots/example_cluster)

## Requirements 

You will need the following packages
* tidyverse
* httr
* jsonlite
* rvest
